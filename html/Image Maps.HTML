HTML Image Maps

An image map allows you to create clickable areas on an image. The <map> tag is used to define an image map, and it is linked to an image by using the usemap attribute on the <img> tag.

Inside the <map> element, you define the clickable areas using one or more <area> tags.

How It Works

  1. Add an <img> tag with a usemap attribute. The value of usemap starts with a hash (#) and is followed by the name of the map.
  2. Create a <map> element with a name attribute that matches the usemap value.
  3. Inside the <map>, add <area> elements to define the clickable regions.

Example: A Workspace Image Map

In this example, we have an image of a desk setup. We will create clickable areas for the laptop, the phone, and the coffee cup.

Click on the computer, the phone, or the coffee cup in the image above.

The Code

<!-- Image with a usemap attribute -->
<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

<!-- The map with a corresponding name -->
<map name="workmap">
  <!-- Clickable rectangular area for the computer -->
  <area shape="rect" coords="34,44,270,350" alt="Computer" href="...">

  <!-- Clickable rectangular area for the phone -->
  <area shape="rect" coords="290,172,333,250" alt="Phone" href="...">

  <!-- Clickable circular area for the coffee cup -->
  <area shape="circle" coords="337,300,44" alt="Coffee" href="...">
</map>

The <area> Tag Attributes

shape

This attribute defines the shape of the clickable area. Possible values are:

  • rect - defines a rectangular region
  • circle - defines a circular region
  • poly - defines a polygonal region
  • default - defines the entire region

coords

This attribute specifies the coordinates of the clickable area. The required values depend on the shape:

  • For shape="rect": coords="x1,y1,x2,y2" where (x1,y1) is the top-left corner and (x2,y2) is the bottom-right corner.
  • For shape="circle": coords="x,y,radius" where (x,y) is the center of the circle and radius is the radius.
  • For shape="poly": coords="x1,y1,x2,y2,...,xn,yn" where each (x,y) pair is a vertex of the polygon.

Coordinates are measured in pixels from the top-left corner of the image (0,0).

href

Specifies the URL that the area links to. If you don't want a link, you can use nohref or set href="#" and handle the click with JavaScript.

alt

Provides alternative text for the area, which is important for accessibility and is displayed if the image cannot be loaded.
