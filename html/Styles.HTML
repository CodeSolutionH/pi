HTML Styles using the style Attribute

The HTML style attribute is used to add styles to an element, such as color, font, size, and more. This is known as an "inline style".

Note: While using the style attribute is a quick way to apply styles, it is generally not recommended for styling entire websites. For better maintainability and separation of concerns, you should use external or internal CSS stylesheets. However, it is useful for quick tests or applying a very specific style to a single element.

The style Attribute Syntax

The syntax is style="property:value;", where property is a CSS property and value is a CSS value.

<tagname style="property:value; property2:value2;">...</tagname>

Examples of Common CSS Properties

Background Color (background-color)

The background-color property defines the background color for an element.

This paragraph has a powder blue background.

<p style="background-color:powderblue;">...</p>

Text Color (color)

The color property defines the text color for an element.

This text is blue.

This text is red.

<p style="color:blue;">...</p>
<p style="color:red;">...</p>

Font Family (font-family)

The font-family property defines the font to be used for an element.

This is in Verdana font.

This is in Courier font.

<p style="font-family:verdana;">...</p>
<p style="font-family:courier;">...</p>

Text Size (font-size)

The font-size property defines the size of the text.

This is normal size (100%).

This is larger (160%).

<p style="font-size:100%;">...</p>
<p style="font-size:160%;">...</p>

Text Alignment (text-align)

The text-align property defines the horizontal alignment of text in an element.

This text is centered.

This text is right-aligned.

<p style="text-align:center;">...</p>
<p style="text-align:right;">...</p>

Combining Multiple Styles

You can apply multiple style properties to one element by separating them with a semicolon.

A Styled Heading

<h2 style="color:tomato; text-align:center; font-family:courier;">...</h2>
