HTML Elements

An HTML element is a component of an HTML document that tells the browser how to structure and display content. Elements are defined by tags, such as <p> for paragraphs or <h1> for headings.

Example:

<p>This is a paragraph element.</p>
<h1>This is a heading element</h1>

How to Use:

  • Start with an opening tag, e.g., <p>
  • Add your content
  • Close with a closing tag, e.g., </p>
