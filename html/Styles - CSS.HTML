Styling HTML with CSS

CSS (Cascading Style Sheets) is used to format the layout of a webpage. It can be used to control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Three Ways to Insert CSS

There are three ways of inserting a style sheet:

  1. External CSS
  2. Internal CSS
  3. Inline CSS

1. External CSS

With an external style sheet, you can change the look of an entire website by changing just one file. Each HTML page must include a link to the style sheet with the <link> tag inside the head section.

This page is linked to an external stylesheet called "Styles - CSS.css". The blue background of this page is set by that external file.

<!-- In your HTML <head> -->
<link rel="stylesheet" href="Styles - CSS.css">
/* In your Styles - CSS.css file */
body {
  background-color: lightblue;
}
h1 {
  color: navy;
}

2. Internal CSS

An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the <style> tag, also inside the head section.

This paragraph has a special style defined in the internal style sheet of this page. Notice the blue border and background. The text color for all paragraphs (including this one) has also been changed to dark green by the internal style sheet, overriding the external one.

<!-- In your HTML <head> -->
<style>
  p {
    color: darkgreen;
  }
  .special-paragraph {
    border: 2px solid darkblue;
  }
</style>

3. Inline CSS

An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

This paragraph has an inline style.

Inline styles have the highest priority and will override both internal and external styles.

<p style="color: red; font-size: 20px;">This paragraph has an inline style.</p>

Cascading Order (Priority)

What style will be used when there is more than one style specified for an HTML element?

The priority of styles is as follows (from highest to lowest):

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section). If selectors are equal, the one that comes last in the code wins.
  3. Browser default

So, an inline style has the highest priority, and will override external and internal styles and browser defaults.
