The HTML class Attribute

The class attribute specifies one or more class names for an HTML element. It is primarily used to point to a class in a style sheet, allowing CSS to select and style specific elements. It can also be used by JavaScript to manipulate elements.

Using the class Attribute

In the example below, we have three <div> elements that all share the same class name, city. This allows us to apply the same CSS styles to all of them at once.

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.

The HTML and CSS Code

HTML:

<div class="city">
    <h2>London</h2>
    <p>London is the capital of England.</p>
</div>

<div class="city">
    <h2>Paris</h2>
    <p>Paris is the capital of France.</p>
</div>

CSS (inside <style> tags or an external file):

.city {
    background-color: tomato;
    color: white;
    border: 2px solid black;
    margin: 20px;
    padding: 20px;
}

The dot (.) in the CSS selector indicates a class name.

Multiple Classes

HTML elements can belong to more than one class. To specify multiple classes, separate the class names with a space.

In this example, the <h2> element has two classes, note and main-text, and will inherit the CSS properties from both.

A Heading with Multiple Classes

This paragraph does not have the special styling.

The HTML and CSS Code

HTML:

<h2 class="note main-text">A Heading with Multiple Classes</h2>

CSS:

.note {
    font-size: 120%;
    color: red;
}

.main-text {
    text-align: center;
}

Different Elements Sharing a Class

Different types of elements can share the same class name and get the same styling.

London

This paragraph also has the 'city' class.

<h2 class="city">London</h2>
<p class="city">This paragraph also has the 'city' class.</p>

Use in JavaScript

The class name can also be used by JavaScript to perform certain tasks for specified elements. For example, you can use document.getElementsByClassName('my-class') to get all elements with a certain class.
