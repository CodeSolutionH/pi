HTML Table Sizes (Width and Height)

By default, the size of a table and its columns is determined automatically by the content inside them. However, you can control the size of the table and its elements using CSS properties like width and height.

Using the old width and height attributes on table elements is not recommended. All styling should be done with CSS.

Table Width

You can set the width of the entire table or individual columns.

Full-Width Table

To make a table stretch to the full width of its container, set the width property to 100%.

FirstnameLastnameAge
JillSmith50
EveJackson94
table {
    width: 100%;
}

Specific Column Widths

You can set the width of a specific column by applying a width to either the <th> or <td> elements in that column, or more efficiently, by using the <col> or <colgroup> elements.

Item Description
Computer A very long description about the computer that will likely overflow the cell unless handled properly with CSS like text-overflow. 
#table-col-width {
    width: 100%;
}

/* Using colgroup is efficient */
#table-col-width .col1 { 
    width: 30%; 
}
#table-col-width .col2 { 
    width: 70%; 
}

/* You also need to handle potential text overflow */
td {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
}

Table Height

You can set the height of table rows by applying the height property to <tr>, <th>, or <td> elements. The height of a row will be determined by the tallest cell within that row.

Example of Row Heights:

Month Savings
January $100
February $150

The Code:

th {
    height: 50px;
}

td {
    height: 70px;
}
