HTML Headings

HTML headings are titles or subtitles that you want to display on a webpage. They are defined with the <h1> to <h6> tags.

<h1> defines the most important heading, while <h6> defines the least important heading.

The Code

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Important: Browsers automatically add some white space (a margin) before and after a heading. The size and boldness of the font decrease from <h1> to <h6>.

Why Headings are Important

  • Search Engine Optimization (SEO): Search engines use headings to index the structure and content of your web pages. A well-structured document with clear headings can rank better in search results.
  • Accessibility: Screen readers use headings to help visually impaired users navigate a web page. A logical heading structure is crucial for accessibility.
  • Readability: Headings break up the text and make the content easier to read and scan for all users.

Best Practices

  • Use only one <h1> per page. It should be the main title of the page content.
  • Use headings in sequential order without skipping levels (e.g., go from an <h2> to an <h3>, not directly to an <h4>).
  • Use headings for structure, not just to make text big or bold. For styling, use CSS properties like font-size and font-weight instead.

Customizing Heading Styles

You can easily change the default appearance of headings using CSS.

<style>
  h1 {
    color: steelblue;
    font-size: 3em; /* 3 times the base font size */
    text-align: center;
  }
  h2 {
    color: #555;
    border-bottom: 1px solid #ccc;
  }
</style>
