HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab. It is also used as the icon for bookmarks, on mobile home screens, and in other places across the web.

How to Add a Favicon

You can add a favicon to your website by adding a <link> element to your HTML's <head> section.

The Code

Place the following line inside the <head> section of your HTML document:

<head>
  <title>My Page Title</title>
  <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>

Attributes Explained:

  • rel="icon": This attribute specifies the relationship between the current document and the linked resource. icon indicates that the resource is a small icon representing the document.
  • type="image/x-icon": This attribute specifies the MIME type of the linked resource. While image/x-icon is the traditional format for .ico files, modern browsers support other formats as well.
  • href="/images/favicon.ico": This attribute provides the URL to the favicon image. It can be a relative path or an absolute URL.

Favicon File Formats

While the .ico format is the most traditional, modern browsers support several other image formats for favicons:

Format MIME Type Notes
ICO image/x-icon Widely supported, can contain multiple sizes.
PNG image/png Good for transparency, widely supported.
GIF image/gif Supports animation, but less common for favicons.
JPEG image/jpeg Not ideal as it doesn't support transparency.
SVG image/svg+xml Scalable vector format, great for high-res screens.

<link rel="icon" type="image/png" href="/images/favicon.png">
