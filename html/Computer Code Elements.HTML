HTML Computer Code Elements

HTML provides several elements for formatting text that represents computer code, keyboard input, variable names, and program output. These are semantic tags that tell the browser what kind of text they contain.

The <code> Element

The <code> element is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.

Example:

A piece of code: var x = 5;

Another piece of code: console.log("Hello, World!");

<p>A piece of code: <code>var x = 5;</code></p>
<p>Another piece of code: <code>console.log("Hello, World!");</code></p>

Note: The <code> element does not preserve extra whitespace and line breaks. To fix this, you can put the <code> element inside a <pre> (preformatted text) element.

<pre>
<code>
function greet() {
  return "Hello!";
}
</code>
</pre>

The <kbd> Element

The <kbd> element is used to define keyboard input. The content inside is displayed in the browser's default monospace font.

Example:

Save the document by pressing Ctrl + S.

To quit, type exit and press Enter.

<p>Save the document by pressing <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>
<p>To quit, type <kbd>exit</kbd> and press <kbd>Enter</kbd>.</p>

The <samp> Element

The <samp> element is used to define sample output from a computer program. The content inside is displayed in the browser's default monospace font.

Example:

If you run the program, you will get this output:

File not found.
Press F1 to continue

<p>If you run the program, you will get this output:</p>
<p><samp>File not found.<br>Press F1 to continue</samp></p>

The <var> Element

The <var> element is used to define a variable in a mathematical expression or a programming context. The content inside is typically displayed in italic.

Example:

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.

In the expression E = mc2, E is the variable for energy.

<p>The area of a triangle is: 1/2 x <var>b</var> x <var>h</var>...</p>
<p>In the expression <var>E</var> = <var>m</var><var>c</var><sup>2</sup>...</p>
