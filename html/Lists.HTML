HTML Lists

Lists are used to group related items together. HTML supports three main types of lists:

  • Unordered List (<ul>): Items are marked with bullets.
  • Ordered List (<ol>): Items are numbered.
  • Description List (<dl>): Items have terms and descriptions.

Unordered List Example:

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Cherry</li>
</ul>
  • Apple
  • Banana
  • Cherry

Ordered List Example:

<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>
  1. First
  2. Second
  3. Third

Description List Example:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

Lists help organize content and make it easier to read. You can nest lists inside each other for more complex structures.
