Advanced HTML Table Styling with CSS

With CSS, you can greatly enhance the appearance of your HTML tables, making them more readable and visually appealing.

Zebra-Striped Tables

Zebra-striping (adding alternating background colors to rows) makes it easier to read data across a table. This is achieved with the :nth-child(even) or :nth-child(odd) selector.

Example:

FirstnameLastnameAge
JillSmith50
EveJackson94
JohnDoe80
PeterJones35

The Code:

<!-- CSS -->
/* Style the table headers */
th {
  background-color: #4CAF50;
  color: white;
}

/* Style every even table row */
tr:nth-child(even) {
  background-color: #f2f2f2;
}

/* Add a hover effect on rows */
tr:hover {
  background-color: #ddd;
}

Vertical Borders

You can create a clean, modern look by using only horizontal borders (or vertical, if you prefer).

Example:

FirstnameLastnameSavings
PeterGriffin$100
LoisGriffin$150
JoeSwanson$300
ClevelandBrown$250

The Code:

<!-- CSS -->
th, td {
  /* Add only a bottom border to cells */
  border-bottom: 1px solid #ddd;
}

th {
  /* Make the header border stronger */
  border-bottom: 2px solid #333;
}

Responsive Tables

A common problem with HTML tables is that they don't naturally work well on small screens. A table with many columns will often overflow the screen width. To make a table responsive, you can wrap it in a container with overflow-x: auto;. This will add a horizontal scrollbar to the table on small screens, allowing the user to scroll to see all the content.

Example:

FirstnameLastnameAgeCityCountry
JillSmith50New YorkUSA
EveJackson94LondonUK

The Code:

<div style="overflow-x:auto;">
  <table>
    ...
  </table>
</div>
