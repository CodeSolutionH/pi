HTML Background Images

A background image can be specified for most HTML elements using the CSS background-image property.

Background Image on an Element

Here, we add a background image to a <div> element. We also specify other background properties to control its behavior.

<style>
.element-with-bg {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: right top;
}
</style>

<div class="element-with-bg" style="height: 300px; border: 1px solid black;">
  This div has a background image.
</div>

This div has a background image.

The image is positioned at the top right.

Background Image on a Page

To set a background image for the entire page, you apply the CSS to the <body> element.

<style>
body {
    background-image: url('background.jpg');
    background-repeat: no-repeat; /* Optional: prevents tiling */
    background-size: cover;       /* Optional: scales the image to cover the entire screen */
}
</style>

(This example is not applied to the whole page to maintain readability, but the code shows how it's done.)

Fullscreen Responsive Background Image

To have an image cover the entire browser window and scale smoothly, you can use the background-size: cover; property combined with background-attachment: fixed; to create a "sticky" or "parallax" effect.

<style>
.fullscreen-bg {
    background-image: url('image.jpg');
    height: 100vh; /* 100% of the viewport height */
    background-position: center;
    background-repeat: no-repeat;
    background-size: cover;
    background-attachment: fixed; /* This creates the parallax effect */
}
</style>

Scroll down to see the fixed background image create a parallax-like effect. The image stays in place while the content scrolls over it.

The key CSS properties are background-attachment: fixed; and background-size: cover;.
