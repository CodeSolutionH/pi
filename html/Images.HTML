HTML Images

Images can improve the design and the appearance of a web page. In HTML, images are defined with the <img> tag.

The <img> tag is an empty element, meaning it contains attributes only and does not have a closing tag.

Basic Image Syntax

To display an image, you need two required attributes:

  • src - Specifies the path (URL) to the image.
  • alt - Specifies an alternate text for the image.

The Code:

<img src="https://www.w3schools.com/html/pic_trulli.jpg" alt="Trulli houses in Alberobello, Italy">

The alt attribute is crucial for accessibility. If an image cannot be displayed (e.g., due to a slow connection, an error in the src path, or if the user uses a screen reader), the value of the alt attribute is displayed instead.

Image Size - width and height

You can use the width and height attributes to specify the size of an image in pixels. It's a good practice to specify both to prevent layout shifts while the page is loading.

The Code:

<img src="img_girl.jpg" alt="Girl in a jacket" width="300" height="400">

Note: You can also use the style attribute to set the width and height with CSS. Using CSS is often more flexible, for example, when creating responsive designs (e.g., style="width:100%;").

Images from Another Server

You can use an absolute URL in the src attribute to link to an image hosted on another website.

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Animated Images (GIFs)

HTML allows you to use animated images, most commonly in the GIF format.

<img src="programming.gif" alt="Computer Man">

Using an Image as a Link

To use an image as a link, simply put the <img> tag inside an <a> (anchor) tag.

<a href="default.asp">
  <img src="smiley.gif" alt="HTML tutorial">
</a>

Common Image Formats

  • JPEG (or JPG): Best for photographs. Good compression, but it's "lossy" (loses quality). Does not support transparency.
  • PNG: Good for graphics with transparency or few colors. "Lossless" compression (no quality loss).
  • GIF: Best for simple animations. Limited to 256 colors. Supports transparency.
  • SVG: Scalable Vector Graphics. An XML-based vector image format. It can be scaled to any size without losing quality. Great for logos and icons.
  • WebP: A modern format developed by Google that provides superior lossless and lossy compression for images on the web. It supports transparency and animation.
