HTML Symbols

Many mathematical, technical, and currency symbols are not present on a normal keyboard. To add these symbols to an HTML page, you can use an HTML entity name or an entity number.

If a character does not have an entity name, you can use its decimal or hexadecimal entity number.

Using HTML Entities for Symbols

For example, to add the Euro symbol (€), you can use the entity name &euro; or the entity number &#8364;.

<p>The price is 50 &euro;.</p>
<p>The price is 50 &#8364;.</p>

Result:

The price is 50 €.

The price is 50 €.

Combining with Diacritical Marks

Some characters can be combined with diacritical marks (like accents) to create new characters. For example, &#769; is a combining acute accent. You can place it after a letter to accent it.

Example: Á (A + combining acute accent)

A&#769;
