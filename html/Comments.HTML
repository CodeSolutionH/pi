HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code. This is useful for leaving notes for yourself or other developers.

Comment Tag Syntax

You can add comments to your HTML source by using the following syntax:

<!-- Write your comments here -->

Notice that there is an exclamation point (!) in the start tag, but not in the end tag.

Examples

1. Commenting Out Text

You can place comments anywhere in your HTML document. Here is a comment placed within a paragraph.

This is a paragraph.

This is another paragraph.

Code:

<p>This is a paragraph.</p>
<!-- This is a comment that will not be displayed -->
<p>This is another paragraph.</p>

2. Commenting Out Code (Debugging)

Comments are also great for debugging HTML. You can comment out lines of code to see how a page renders without them, helping you find errors.

This paragraph is visible.

This paragraph is also visible.

Code:

<p>This paragraph is visible.</p>

<!--
<p>This paragraph is commented out and will not be seen.</p>
<img src="some_image.jpg" alt="A hidden image">
-->

<p>This paragraph is also visible.</p>

3. Conditional Comments (for Internet Explorer)

In the past, special comments were used to target specific versions of the Internet Explorer (IE) browser. This is now largely obsolete but is good to recognize in older codebases.

Code Example (Not rendered by modern browsers):

<!--[if IE 9]>
    <p>This content is only displayed in Internet Explorer 9.</p>
    <link rel="stylesheet" type="text/css" href="ie9.css">
<![endif]-->

Important Notes

  • Comments are not displayed by the browser.
  • They can be used to explain your code, which can help you when you edit the source code at a later date.
  • You cannot have a comment inside another comment.
  • Comments are visible to anyone who views the page source code. Do not put sensitive information in comments.
