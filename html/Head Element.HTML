The HTML <head> Element

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

Metadata is data about the HTML document. It is not displayed on the page itself, but is used by browsers, search engines, and other web services.

Note: The content inside the <head> element is not visible in the main browser window.

Anatomy of an HTML Document

<!DOCTYPE html>
<html lang="en">

<head>
  <!-- Metadata goes here -->
</head>

<body>
  <!-- Visible page content goes here -->
</body>

</html>

Common Tags Found in the <head>

The following tags are commonly placed inside the <head> section:

1. The <title> Tag

The <title> tag is required in all HTML documents. It defines the title of the document, which is shown in the browser's title bar or in the page's tab. It is also used by search engines for search results.

<title>A Meaningful Page Title</title>

2. The <style> Tag

The <style> tag is used to define internal CSS styles for a single HTML page.

<style>
  body {
    background-color: linen;
  }
  h1 {
    color: maroon;
  }
</style>

3. The <link> Tag

The <link> tag is most often used to link to external style sheets (CSS files). It can also be used to add a favicon.

<!-- Linking to an external CSS file -->
<link rel="stylesheet" href="mystyle.css">

<!-- Adding a favicon -->
<link rel="icon" type="image/x-icon" href="favicon.ico">

4. The <meta> Tag

The <meta> tag is used to specify character set, page description, keywords, author of the document, and viewport settings.

<!-- Specify the character encoding for the document -->
<meta charset="UTF-8">

<!-- Provide a description for search engines -->
<meta name="description" content="Free Web tutorials">

<!-- Set the viewport to make the website responsive -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

5. The <script> Tag

The <script> tag is used to embed client-side JavaScripts or to link to external JavaScript files. While it can be placed in the <head>, it's often better for performance to place it at the end of the <body> section.

<!-- Linking to an external script file -->
<script src="myscript.js"></script>

6. The <base> Tag

The <base> tag specifies the base URL and/or target for all relative URLs in a page. A document can only have one <base> element.

<base href="https://www.w3schools.com/" target="_blank">

Example of a Complete <head> Section

<head>
  <meta charset="UTF-8">
  <meta name="description" content="An example page">
  <meta name="keywords" content="HTML, Head, Meta, Title">
  <meta name="author" content="John Doe">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Example Page</title>
  <link rel="stylesheet" href="styles.css">
  <link rel="icon" type="image/png" href="favicon.png">
</head>
