HTML Emojis

Emojis are characters from the UTF-8 character set. Emojis look like images, or icons, but they are not. They are letters (characters) from the UTF-8 (Unicode) character set.

Using Emojis in HTML

To use emojis in an HTML file, you need to ensure the character encoding is set to UTF-8. This is done by adding the <meta charset="UTF-8"> tag inside the <head> element.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page with Emojis</title>
</head>
<body>

    <h1>My First Emoji &#128512;</h1>

</body>
</html>

How to Add Emojis

You can include emojis in your HTML in a few ways:

  1. Copy and Paste: The easiest way is to simply copy the emoji from a source like Emojipedia or your operating system's character map and paste it directly into your HTML code.
  2. Using Entity Numbers: You can use the emoji's decimal or hexadecimal reference number.

Are Emojis Supported Everywhere?

Emojis are supported by all major operating systems like iOS, Android, macOS, and Windows. However, the visual appearance of an emoji can vary depending on the platform and browser, as each vendor may have its own emoji designs.
