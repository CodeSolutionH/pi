HTML Quotation and Citation Elements

HTML provides several elements for marking up quotations and citations, giving them semantic meaning.

<blockquote> for Long Quotations

The <blockquote> element is used to define a section that is quoted from another source. Browsers usually indent <blockquote> elements.

The cite attribute can be used to provide a URL to the source of the quotation.

Example:

Here is a quote from the Mozilla Developer Network (MDN):

The HTML <blockquote> Element (or HTML Block Quotation Element) indicates that the enclosed text is an extended quotation.

The Code:

<blockquote cite="https://developer.mozilla.org/...">
  The HTML <blockquote> Element ... is an extended quotation.
</blockquote>

<q> for Short, Inline Quotations

The <q> tag defines a short quotation. Browsers normally insert quotation marks around the quotation.

Example:

Steve Jobs once said, Design is not just what it looks like and feels like. Design is how it works.

The Code:

<p>Steve Jobs once said, <q>Design is not just what it looks like and feels like. Design is how it works.</q></p>

<abbr> for Abbreviations

The <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Marking up abbreviations can give useful information to browsers, translation systems, and search-engines. The optional title attribute can provide the full expansion of the abbreviation, which is shown as a tooltip on hover.

Example:

The WHO was founded in 1948.

I love to code in HTML and CSS.

The Code:

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

<cite> for Citing a Work Title

The <cite> tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.). The text in the <cite> element usually renders in italic.

Example:

My favorite painting is The Starry Night by Vincent van Gogh.

I highly recommend reading To Kill a Mockingbird.

The Code:

<p>My favorite painting is <cite>The Starry Night</cite> by Vincent van Gogh.</p>

<bdo> for Bi-Directional Override

The <bdo> tag is used to override the current text direction. This is useful for languages that are written from right-to-left, like Arabic.

Example:

This text will be written from left to right.

This text will be written from right to left.

The Code:

<p><bdo dir="rtl">This text will be written from right to left.</bdo></p>
