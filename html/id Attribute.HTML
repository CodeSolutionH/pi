The HTML id Attribute

The id attribute specifies a unique identifier for an HTML element. The value of the id attribute must be unique within the entire HTML document.

Key Rule: No two elements in a document can have the same ID.

Using the id Attribute

The id attribute has two primary uses:

  1. As a selector for CSS to apply a unique style to a specific element.
  2. As a hook for JavaScript to access and manipulate a specific element.

1. Using id with CSS

CSS can select an element with a specific ID by using the hash (#) character, followed by the ID name.

A Uniquely Styled Header

This header has a unique style applied to it via its ID.

The HTML and CSS Code

HTML:

<h1 id="myHeader">A Uniquely Styled Header</h1>

CSS (inside <style> tags or an external file):

#myHeader {
    background-color: lightblue;
    color: black;
    padding: 40px;
    text-align: center;
}

2. Using id with JavaScript

JavaScript can use the getElementById() method to access an element with a specific ID and manipulate it.

Click the button to change this text.

The HTML and JavaScript Code

HTML:

<p id="demo">Click the button to change this text.</p>
<button onclick="changeText()">Change Text</button>

JavaScript:

function changeText() {
  const element = document.getElementById("demo");
  element.innerHTML = "Text has been changed!";
  element.style.color = "red";
}

3. Using id for Bookmark Links

HTML allows you to create a link that jumps to a specific section (a "bookmark") on the same page. This is done by giving an ID to the target element and creating a link with an href value of # followed by the ID.

Jump to Section 2











Section 2

You have arrived at section 2!

<a href="#section2">Jump to Section 2</a>

...

<h2 id="section2">Section 2</h2>

class vs. id

Attribute Rule Usage
class Can be used on multiple elements. An element can have multiple classes. Used for styling groups of elements.
id Must be unique. An element can only have one ID. Used for styling a single, unique element or for JavaScript manipulation.
