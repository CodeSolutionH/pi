HTML Table Colspan & Rowspan

HTML tables can have cells that span over multiple rows or columns. This is achieved using the colspan and rowspan attributes on <th> or <td> elements.

The colspan Attribute

The colspan attribute defines the number of columns a cell should span (merge across).

Example:

In this table, the top header cell ("Name") spans two columns.

Name Age
Jill Smith 50
Eve Jackson 94

The Code:

<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>

The rowspan Attribute

The rowspan attribute defines the number of rows a cell should span (merge down).

Example:

In this table, the first cell ("Telephone") spans two rows.

Name: Bill Gates
Telephone: 555 77 854
555 77 855

The Code:

<table>
  <tr>
    <th>Name:</th>
    <td>Bill Gates</td>
  </tr>
  <tr>
    <th rowspan="2">Telephone:</th>
    <td>555 77 854</td>
  </tr>
  <tr>
    <td>555 77 855</td>
  </tr>
</table>

Complex Example: Both colspan and rowspan

You can combine both attributes to create more complex table structures, such as for timetables or calendars.

Monday Tuesday Wednesday
Morning Math Science History
Afternoon English
Evening Free Time
