HTML Table Borders

By default, HTML tables are displayed without borders. To add borders, you should use CSS.

The Old Way: The border Attribute (Not Recommended)

In older versions of HTML, you could use the border attribute on the <table> tag to add borders.

<table border="1">
  ...
</table>

While most browsers still support this, it is not recommended. Styling should be done with CSS. This method often results in an undesirable double-border effect.

The Modern Way: Using CSS

The modern and recommended way to add borders to tables is by using the CSS border property.

Single Line Borders with border-collapse

To get clean, single-line borders, you first add a border to the <th> and <td> elements. Then, you add the border-collapse: collapse; property to the main <table> element. This collapses the borders into a single border.

Example:

JillSmith50
EveJackson94

The Code:

<!-- HTML -->
<table>
  ...
</table>

<!-- CSS -->
<style>
table {
  border-collapse: collapse; /* Collapses borders into a single border */
  width: 100%;
}

th, td {
  border: 1px solid #ddd; /* Adds a border to all table cells and headers */
  padding: 8px;
  text-align: left;
}

th {
  background-color: #f2f2f2; /* Adds a background to headers */
}
</style>

Alternative Styling: Bottom Borders Only

For a more modern, minimalist look, you can style the table with only horizontal borders.

Example:

FirstnameLastnameAge
JillSmith50
EveJackson94

The Code:

<!-- CSS -->
th, td {
  border: none; /* Remove all borders first */
  border-bottom: 1px solid #ddd; /* Add a bottom border to cells */
}

th {
  border-bottom: 2px solid #333; /* Make the header border thicker */
}
