The HTML Page Title (<title>)

The <title> element defines the title of the HTML document. It is a required tag in every HTML page.

The title element is placed within the <head> section of the HTML document.

Where is the Page Title Used?

The content of the <title> tag is not displayed on the web page itself. Instead, it is used in several other places:

1. Browser Tab / Title Bar

The most visible use of the title is in the browser tab or the title bar of the browser window. Look at the tab for this page to see a live example.

2. Search Engine Results Pages (SERP)

Search engines, like Google, typically use the page title as the main clickable headline for a given search result. A well-written, descriptive title is crucial for Search Engine Optimization (SEO).

When a user bookmarks your page, the page title is used as the default name for that bookmark.

How to Write a Good Title

  • Be Descriptive and Concise: A good title accurately describes the page's content in a brief manner. Aim for under 60 characters, as Google typically displays the first 50-60 characters of a title tag.
  • Be Unique: Every page on your website should have a unique title. This helps search engines (and users) distinguish between pages.
  • Use Keywords Wisely: Place important keywords near the beginning of the title, but avoid "keyword stuffing" (e.g., "Buy Shoes, Best Shoes, Cheap Shoes, Shoes for Sale"). Write for humans first.
  • Brand Your Titles: It's a common practice to include your site or brand name at the end of the title, separated by a pipe (|) or a hyphen (-). For example: HTML Page Title - MyWebsite.

The Code

The code for a title is very simple. It goes inside the <head> section.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>How to Write a Great Page Title</title>
</head>
<body>
    <h1>Page Content</h1>
    <p>This is the content visible on the page.</p>
</body>
</html>
