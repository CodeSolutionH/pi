HTML Iframes

An HTML iframe is used to display a web page within a web page. The <iframe> tag specifies an inline frame.

Basic Iframe Syntax

To create an iframe, you use the <iframe> tag and specify the URL of the page to embed in the src attribute.

<iframe src="https://www.w3schools.com"></iframe>

Iframe Attributes

height and width

You can use the height and width attributes to specify the size of the iframe in pixels or percentages.

<iframe src="..." height="200" width="300"></iframe>

title

The title attribute is important for accessibility. Screen readers use it to read what the content of the iframe is.

<iframe src="..." title="A description of the iframe content"></iframe>

style (Removing the border)

By default, an iframe has a border around it. You can remove this using CSS and the style attribute.

<iframe src="..." style="border:none;"></iframe>

Using an Iframe as a Link Target

An iframe can be used as the target frame for a link. This means that when a user clicks a link, the linked document opens inside the iframe.

To do this, the target attribute of the link must refer to the name attribute of the iframe.

Code:

<p><a href="https://www.w3schools.com" target="iframe_a">W3Schools.com</a></p>

Security Note

Many websites, including major ones like Google and Facebook, do not allow themselves to be embedded in iframes. They do this by sending an HTTP header called X-Frame-Options set to DENY or SAMEORIGIN to prevent "clickjacking" attacks.
