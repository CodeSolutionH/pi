HTML Links, Colors, and Bookmarks

Links are found in nearly all web pages. Links allow users to click their way from page to page.

HTML Links - The <a> Tag

The HTML <a> tag defines a hyperlink. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

<a href="https://www.google.com">Visit Google</a>
Visit Google

The target Attribute

The target attribute specifies where to open the linked document:

  • _self: Default. Opens the document in the same window/tab as it was clicked.
  • _blank: Opens the document in a new window or tab.
  • _parent: Opens the document in the parent frame.
  • _top: Opens the document in the full body of the window.
<a href="https://www.google.com" target="_blank">Visit Google in a new tab</a>
Visit Google in a new tab

Link Colors

By default, a link will appear like this (in most browsers):

  • An unvisited link is underlined and blue.
  • A visited link is underlined and purple.
  • An active link is underlined and red.

You can change the default colors using CSS pseudo-classes:

a:link { /* Style for a normal, unvisited link */
  color: green;
}
a:visited { /* Style for a link the user has visited */
  color: pink;
}
a:hover { /* Style for a link when the user mouses over it */
  color: red;
  text-decoration: none;
}
a:active { /* Style for a link at the moment it is clicked */
  color: yellow;
}
This is a custom styled link. Try hovering and clicking it.

Link as a Button

You can also use CSS to style a link to look like a button, which is a common practice in web design.

I am a Link Button 
<a href="#" class="link-button">I am a Link Button</a>

<style>
.link-button {
    background-color: #28a745;
    color: white;
    padding: 10px 20px;
    text-decoration: none;
    border-radius: 5px;
}
</style>

HTML Link Bookmarks

HTML bookmarks allow users to jump to a specific part of a web page. This is useful for long pages.

Step 1: Create the bookmark by giving an id to the element you want to jump to.

<h2 id="C4">Chapter 4</h2>

Step 2: Create a link that points to that id using a hash character (#).

<a href="#C4">Jump to Chapter 4</a>
