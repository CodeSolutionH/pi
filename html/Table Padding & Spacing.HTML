HTML Table Padding & Spacing

By default, table cells have very little padding and no spacing between their borders. To control the space inside and around the cells, you should use CSS.

The Old Way: cellpadding and cellspacing (Not Recommended)

In older HTML, the cellpadding and cellspacing attributes were used on the <table> tag. These attributes are now obsolete and should not be used. Styling should be done with CSS.

  • cellpadding: Specified the space between the cell content and its borders.
  • cellspacing: Specified the space between cells.
FirstnameLastname
PeterGriffin
<table cellpadding="10" cellspacing="5"> ... </table> <!-- Not recommended -->

The Modern Way: Using CSS

The modern approach uses the CSS padding and border-spacing properties.

1. Cell Padding with CSS padding

To control the space between the cell content and the cell border, use the padding property on <th> and <td> elements.

Example:

FirstnameLastnameAge
JillSmith50
EveJackson94

The Code:

<!-- CSS -->
th, td {
  padding: 15px;
  text-align: left;
}

2. Cell Spacing with CSS border-spacing

To control the distance between the borders of adjacent cells, use the border-spacing property on the <table> element.

Note: The border-spacing property only has an effect if the table's border-collapse property is set to separate.

Example:

FirstnameLastname
PeterGriffin
LoisGriffin

The Code:

<!-- CSS -->
table {
  border-collapse: separate;
  border-spacing: 10px 5px; /* 10px horizontal, 5px vertical */
}
