HTML Paragraphs (<p>)

The HTML <p> element defines a paragraph. Paragraphs are block-level elements and are used to structure textual content.

Browsers automatically add some white space (a margin) before and after each <p> element. This separates them visually from other content.

Basic Paragraph Example

This is the first paragraph. It contains some text that introduces the concept of paragraphs in HTML. It is a block of text separated from other blocks.

This is the second paragraph. Notice how there is a space between this paragraph and the one above it. This is the default browser styling for the <p> tag.

The Code:

<p>This is the first paragraph...</p>
<p>This is the second paragraph...</p>

How Browsers Handle Paragraphs

You cannot be sure how HTML will be displayed. Large or small screens, and resized windows will create different results. With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code.

The browser will automatically remove any extra spaces and lines when the page is displayed:

Example of Collapsed Whitespace:

This paragraph contains a lot of spaces and many line breaks in the source code, but the browser ignores it.

The Code:

<p>
    This paragraph
    contains         a lot of spaces
    and
    many           line breaks in the source code,
    but the        browser ignores it.
</p>

The Horizontal Rule (<hr>)

The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule. It is used to separate content (or define a change) in an HTML page.

The <hr> tag is an empty tag, which means that it has no end tag.

This is some text.

This is some other text, separated by a horizontal rule.

The Code:

<p>This is some text.</p>
<hr>
<p>This is some other text...</p>

The Line Break (<br>)

The HTML <br> element defines a line break. Use <br> if you want a line break (a new line) without starting a new paragraph.

Like <hr>, the <br> tag is an empty tag.

This is a
paragraph with
line breaks.

The Code:

<p>This is a<br>paragraph with<br>line breaks.</p>

The <pre> Element for Preformatted Text

If you need to preserve whitespace and line breaks exactly as they are written in the source code, use the <pre> tag. This is often used for displaying code snippets.

Text in a pre element
is displayed in a fixed-width
font, and it preserves
both      spaces and
line breaks.
