Using JavaScript in HTML

JavaScript is the world's most popular programming language and is the scripting language for the Web. It can be used to make web pages interactive, dynamic, and more functional.

JavaScript is inserted into HTML using the <script> tag.

The <script> Tag

You can place JavaScript code directly inside the <script> tag or link to an external JavaScript file.

1. Internal JavaScript

You can place your script within the <head> or <body> section of your HTML file.

Example: Changing HTML Content

This button uses JavaScript to change the content of the paragraph below it.

This is a demonstration paragraph.

Code:

<p id="demo">This is a demonstration paragraph.</p>

<button type="button" 
  onclick="document.getElementById('demo').innerHTML = 'Hello JavaScript!'">
  Click Me
</button>

Example: Changing an Image

JavaScript can also be used to change HTML attribute values, like the src of an image.


Code:

<img id="myImage" src="pic_bulboff.gif">

<button onclick="document.getElementById('myImage').src='pic_bulbon.gif'">Turn on</button>
<button onclick="document.getElementById('myImage').src='pic_bulboff.gif'">Turn off</button>

2. External JavaScript

For more complex scripts, it is best practice to place them in external JavaScript files. This makes the code more organized, reusable, and easier to maintain. It also improves page load speed as browsers can cache the external files.

You link to an external script file using the src attribute in the <script> tag.

HTML File (index.html):

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>

  <h1>Using an External Script</h1>
  <p id="external-demo">A Paragraph</p>
  <button type="button" onclick="myFunction()">Try it</button>

  <script src="myScript.js"></script>

</body>
</html>

JavaScript File (myScript.js):

function myFunction() {
  document.getElementById("external-demo").innerHTML = "Paragraph changed from an external file.";
}

Where to Place the <script> Tag

Scripts can be placed in the <body> or in the <head> section of an HTML page, or in both.

  • In <head>: Scripts placed here are loaded before the HTML content is parsed. This can be useful if the script needs to run before the page is displayed, but it can also slow down the initial page rendering.
  • In <body> (at the end): This is the recommended approach for most scripts. Placing scripts at the end of the <body> allows the HTML content to be parsed and displayed to the user first, improving perceived performance. The script can then load and execute without blocking the content.
