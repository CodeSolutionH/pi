The HTML <picture> Element

Description

The HTML <picture> element is a container used to specify multiple <source> elements for a specific <img>. The browser will choose the most suitable source according to the current viewport's width, pixel-density, and other factors, and display the image.

This is particularly useful for providing different image versions for different screen sizes (art direction) or for providing different image formats (like WebP or AVIF) for browsers that support them, with a fallback for those that don't.

Syntax

<picture>
   <source media="(min-width: 650px)" srcset="img_pink_flowers.jpg">
   <source media="(min-width: 465px)" srcset="img_white_flower.jpg">
   <img src="img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
</picture>

Example: Art Direction

In this example, the browser will display a different image based on the viewport width. If the width is 650px or more, it shows pink flowers. If it's 465px or more, it shows a white flower. Otherwise, it shows orange flowers.

Resize the browser window to see the effect.

Example: Format Selection

Here, the browser can choose a modern, more efficient image format like WebP if it supports it, and fall back to a standard JPG if it doesn't.

<picture>
   <source srcset="image.webp" type="image/webp">
   <source srcset="image.jpg" type="image/jpeg">
   <img src="image.jpg" alt="Description of image">
</picture>

This helps in optimizing website performance by serving smaller image files to capable browsers.
