HTML Colors: RGB, HEX, and HSL

Colors in HTML can be specified using predefined color names, or as RGB, HEX, HSL, or RGBA / HSLA values. They are most often used in CSS to style elements.

Color Names

HTML supports 140 standard color names. These are simple, human-readable keywords.

<p style="color: Tomato;">This text is Tomato red.</p>
<p style="background-color: DodgerBlue;">This paragraph has a DodgerBlue background.</p>
This text is Tomato red.
This paragraph has a DodgerBlue background.

HEX (Hexadecimal) Color

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color. Each value can range from 00 to FF (which is 0 to 255 in decimal).

For example, #ff0000 is displayed as red, because red is set to its highest value (ff) and the others are set to the lowest value (00).

<h2 style="color: #ff6347;">This heading is Tomato red in HEX</h2>
<h2 style="color: #4169e1;">This heading is RoyalBlue in HEX</h2>
#ff6347
#4169e1
#3cb371

RGB (Red, Green, Blue) Color

RGB color values are specified with the rgb(red, green, blue) function. Each parameter defines the intensity of the color as an integer between 0 and 255.

For example, rgb(255, 0, 0) is displayed as red, because red is set to its highest value (255) and the others are set to 0.

<h2 style="color: rgb(255, 99, 71);">This heading is Tomato red in RGB</h2>
<h2 style="color: rgb(65, 105, 225);">This heading is RoyalBlue in RGB</h2>
rgb(255, 99, 71)
rgb(65, 105, 225)
rgb(60, 179, 113)

RGBA Color

RGBA color values are an extension of RGB with an alpha channel (opacity). It is specified with rgba(red, green, blue, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

<div style="background-color: rgba(255, 99, 71, 0.5);">Semi-transparent Tomato</div>
rgba(..., 0.5)

HSL (Hue, Saturation, Lightness) Color

HSL color values are specified with the hsl(hue, saturation, lightness) function.

  • Hue: A degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
  • Saturation: A percentage value. 0% means a shade of gray, and 100% is the full color.
  • Lightness: A percentage value. 0% is black, 50% is normal, and 100% is white.
<h2 style="color: hsl(9, 100%, 64%);">This heading is Tomato red in HSL</h2>
<h2 style="color: hsl(225, 73%, 57%);">This heading is RoyalBlue in HSL</h2>
hsl(9, 100%, 64%)
hsl(225, 73%, 57%)
hsl(147, 50%, 47%)

HSLA Color

HSLA is just like HSL but adds an alpha channel for opacity, e.g., hsla(9, 100%, 64%, 0.5).
