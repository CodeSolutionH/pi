HTML Other List Types

Besides the common ordered (<ol>) and unordered (<ul>) lists, HTML supports description lists. Additionally, CSS can be used to style lists in many different ways, such as creating horizontal navigation menus.

HTML Description Lists (<dl>)

A description list is a list of terms, with a description of each term. This is also known as a definition list.

  • The <dl> tag defines the description list.
  • The <dt> tag defines the term (the name).
  • The <dd> tag describes or defines the term.

Example:

Coffee
- A hot, dark beverage made from roasted coffee beans.
Milk
- A white, cold beverage produced by mammals.
Tea
- A hot beverage made by infusing dried, crushed leaves in hot water.

The Code:

<dl>
  <dt>Coffee</dt>
  <dd>- A hot, dark beverage made from roasted coffee beans.</dd>
  <dt>Milk</dt>
  <dd>- A white, cold beverage produced by mammals.</dd>
  <dt>Tea</dt>
  <dd>- A hot beverage made by infusing dried, crushed leaves in hot water.</dd>
</dl>

Styling Lists with CSS

CSS provides many properties to style lists. You can change the marker type, use images for markers, or remove them entirely.

Horizontal List for Navigation

One of the most common uses of styled lists is to create a horizontal navigation menu from a standard HTML unordered list.

Example:

This navigation bar is an unordered list styled with CSS:

  • Home
  • News
  • Contact
  • About

The Code:

HTML:

<ul class="horizontal-list">
  <li><a href="#home">Home</a></li>
  <li><a href="#news">News</a></li>
  <li><a href="#contact">Contact</a></li>
  <li><a href="#about">About</a></li>
</ul>

CSS:

.horizontal-list {
    list-style-type: none; /* Remove bullets */
    margin: 0;
    padding: 0;
    overflow: hidden; /* Contain floats */
    background-color: #333;
}

.horizontal-list li {
    float: left; /* Make list items appear side-by-side */
}

.horizontal-list li a {
    display: block;
    color: white;
    text-align: center;
    padding: 14px 16px;
    text-decoration: none;
}

.horizontal-list li a:hover {
    background-color: #111;
}
