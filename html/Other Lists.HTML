HTML Description Lists

A description list is used for pairs of terms and descriptions. Use the <dl> element for the list, <dt> for each term, and <dd> for each description.

Example:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
HTML
HyperText Markup Language
CSS
Cascading Style Sheets

Best Practices:

  • Use description lists for glossaries, definitions, or metadata.
  • Pair each <dt> with a <dd> for clarity.
