HTML <colgroup> and <col> Elements

The <colgroup> element is used to group one or more columns in a table for formatting. It is a container for one or more <col> elements.

This is useful for applying styles to entire columns, instead of having to repeat the styles for each cell in that column.

The <colgroup> tag must be a child of a <table> element, after any <caption> elements and before any <thead>, <tbody>, <tfoot>, and <tr> elements.

Styling Columns

In this example, we use <colgroup> and <col> to style the first two columns and the third column of a table differently.

Example:

ISBN Title Price
3476896 My first HTML $53
5869207 My first CSS $49

The Code:

<table id="table-colgroup">
  <colgroup>
    <!-- The "span" attribute makes this apply to the first 2 columns -->
    <col span="2" class="group1">
    <!-- This applies to the 3rd column -->
    <col class="group2">
  </colgroup>
  <tr>
    <th>ISBN</th>
    <th>Title</th>
    <th>Price</th>
  </tr>
  ...
</table>

<style>
  #table-colgroup .group1 { background-color: #D6EEEE; }
  #table-colgroup .group2 { background-color: #F2D7D5; }
</style>

The <col> element is an empty tag. The span attribute specifies how many consecutive columns the <col> element should span.

Hiding Columns

You can use the visibility: collapse; CSS property on a <col> element to hide a column from the table. Note that this may not be fully supported in all browsers, and using other methods might be more reliable if you need to hide columns dynamically.

Example:

First Name Last Name (Hidden) Age
John Doe 30
Jane Smith 25

The Code:

<table id="table-visibility">
  <colgroup>
    <col>
    <col class="hidden-col">
    <col>
  </colgroup>
  ...
</table>

<style>
  #table-visibility col.hidden-col {
    visibility: collapse;
  }
</style>

Permitted CSS Properties

Not all CSS properties can be applied to columns. You can only use properties that affect the column as a whole:

  • width
  • border
  • background-color
  • visibility
