Responsive Web Design

Responsive Web Design (RWD) is about creating web pages that look good on all devices (desktops, tablets, and phones). A responsive design will automatically adjust for different screen sizes and viewports.

Key Idea: The website should adapt to the user's device, not the other way around.

1. The Responsive Viewport

To create a responsive website, you must add the following <meta> tag to all your web pages. This gives the browser instructions on how to control the page's dimensions and scaling.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • width=device-width sets the width of the page to follow the screen-width of the device.
  • initial-scale=1.0 sets the initial zoom level when the page is first loaded by the browser.

Without this meta tag, web pages designed for desktop will be shrunk down on mobile, making them unreadable without zooming.

2. Responsive Images

A common way to make images responsive is to use the CSS max-width property. If the max-width property is set to 100%, the image will scale down if it has to, but never scale up to be larger than its original size.

img {
    max-width: 100%;
    height: auto;
}

3. Responsive Text Size

Text size can be set with a responsive unit, like "vw" (viewport width). This way the text size will follow the size of the browser window.

h1 {
    font-size: 10vw; /* 10% of viewport width */
}

4. Media Queries

Media queries are a key part of responsive design. They allow you to apply different CSS styles for different devices based on characteristics like screen width, height, orientation, etc.

In this example, the background color of the body will be lightblue on screens that are 600px wide or wider. On smaller screens, it will be the default.

/* For desktop: */
.col-1 {width: 8.33%;}
.col-2 {width: 16.66%;}
/* ... and so on ... */

/* For mobile phones: */
@media only screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
  [class*="col-"] { /* All columns become full-width */
    width: 100%;
  }
}

5. Modern Layout Modules: Flexbox and Grid

Modern CSS layout modules like Flexbox and CSS Grid are inherently responsive and make it much easier to build flexible layouts that adapt to different screen sizes without complex media queries.
