HTML Ordered Lists (<ol>)

An ordered list is a list where the order of the items is important. It starts with the <ol> tag, and each list item starts with the <li> tag. By default, the list items are marked with numbers.

Basic Ordered List

Morning Routine

  1. Wake up
  2. Make the bed
  3. Brush teeth
  4. Exercise
<ol>
  <li>Wake up</li>
  <li>Make the bed</li>
  <li>Brush teeth</li>
  <li>Exercise</li>
</ol>

The type Attribute

The type attribute of the <ol> tag can be used to change the type of the list item marker.

Type: Numbers (default)

  1. Apples
  2. Bananas
  3. Oranges
<ol type="1"> ... </ol>

Type: Uppercase Letters

  1. Apples
  2. Bananas
  3. Oranges
<ol type="A"> ... </ol>

Type: Lowercase Letters

  1. Apples
  2. Bananas
  3. Oranges
<ol type="a"> ... </ol>

Type: Uppercase Roman Numerals

  1. Apples
  2. Bananas
  3. Oranges
<ol type="I"> ... </ol>

Type: Lowercase Roman Numerals

  1. Apples
  2. Bananas
  3. Oranges
<ol type="i"> ... </ol>

The start Attribute

The start attribute specifies the starting value of the first list item. The following list items will increment from that number.

Continue Countdown

  1. Five
  2. Six
  3. Seven
<ol start="5">
  <li>Five</li>
  <li>Six</li>
  <li>Seven</li>
</ol>

The reversed Attribute

The reversed attribute is a boolean attribute that specifies that the list order should be descending (e.g., 3, 2, 1).

Countdown!

  1. Three
  2. Two
  3. One
  4. Liftoff!
<ol reversed>
  <li>Three</li>
  <li>Two</li>
  <li>One</li>
</ol>
