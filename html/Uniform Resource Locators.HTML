HTML Uniform Resource Locators (URLs)

A URL is the address of a resource on the internet. It's how browsers find and retrieve web pages, images, and other files.

URL Structure:

scheme://prefix.domain:port/path/filename
  • scheme: The type of service (e.g., http, https, ftp).
  • prefix: The domain prefix (e.g., www).
  • domain: The domain name (e.g., google.com).
  • port: The port number on the server (e.g., 80, 443).
  • path: The path to the resource on the server.
  • filename: The name of the resource.

URL Encoding:

URLs can only contain ASCII characters. Non-ASCII characters must be converted into a valid format. For example, spaces are replaced with %20.
