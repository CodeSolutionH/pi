HTML Layout Elements and Techniques

Websites often display content in multiple columns, like a magazine or a newspaper. Creating these layouts is a fundamental part of web design.

HTML5 Semantic Layout Elements

HTML5 introduced several semantic elements that define the different parts of a web page. Using them makes your HTML easier to read and understand for both developers and search engines.

  • <header> - Defines a header for a document or a section.
  • <nav> - Defines a container for navigation links.
  • <section> - Defines a section in a document.
  • <article> - Defines an independent, self-contained article.
  • <aside> - Defines content aside from the page content (like a sidebar).
  • <footer> - Defines a footer for a document or a section.
  • <details> - Defines additional details that the user can view or hide.
  • <summary> - Defines a visible heading for a <details> element.

Example of a Semantic Layout Structure:

<header> - Page Header
<nav> - Navigation Bar
<section>
<article> - Main Content
<article> - Another Piece of Content
<aside> - Sidebar
<footer> - Page Footer
<body>
  <header>...</header>
  <nav>...</nav>
  <section>
    <article>...</article>
  </section>
  <aside>...</aside>
  <footer>...</footer>
</body>

Layout Techniques

There are several ways to create multicolumn layouts, each with its pros and cons. Modern web design primarily uses Flexbox and Grid.

1. CSS Flexbox

Flexbox (the Flexible Box Layout Module) is a one-dimensional layout model that offers an easy and clean way to arrange items in a row or a column. It's excellent for component layouts and overall page layouts.

Example: A Main Content and Sidebar Layout

Main Content (flex: 70%)
Sidebar (flex: 30%)

CSS Code:

.flex-container {
    display: flex; /* This enables flexbox */
}

.flex-main {
    flex: 70%; /* Main content takes 70% of the width */
}

.flex-aside {
    flex: 30%; /* Sidebar takes 30% */
}

2. CSS Grid Layout

CSS Grid is a two-dimensional layout system, meaning it can handle both columns and rows. It is the most powerful layout system available in CSS and is perfect for complex, grid-based page layouts.

3. Other Techniques (Older)

  • CSS Floats: A very common technique for many years, but it can be tricky to use correctly (e.g., requires clearing floats). Flexbox and Grid are now preferred.
  • CSS Positioning: Using position: absolute; or position: relative; can create layouts, but it's not ideal for main page structure as it can make content overlap and be less responsive.
  • HTML Tables: In the early days of the web, tables were often used for layout. This is now considered bad practice. Tables should only be used for displaying tabular data.
