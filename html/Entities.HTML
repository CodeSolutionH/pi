HTML Entities

In HTML, some characters are reserved, like the less-than (<) and greater-than (>) signs, which are used to define tags. To display these characters as text, you must use HTML entities.

An entity is a piece of text ("string") that begins with an ampersand (&) and ends with a semicolon (;).

Commonly Used HTML Entities

Here are some of the most frequently used entities in HTML.

Less-than and Greater-than Signs

To display the less-than sign, you must use an entity because it would otherwise be interpreted as the start of a new HTML tag.

<p>The &lt;p&gt; tag defines a paragraph.</p>

Result:

The <p> tag defines a paragraph.

Non-Breaking Space

A common character entity used in HTML is the non-breaking space: &nbsp;. It looks like a regular space, but it prevents a line break from happening at its position. Two or more regular spaces in a row will be collapsed into one by the browser, but multiple &nbsp; entities will not.

<p>10&nbsp;km/h</p> <!-- Prevents breaking between 10 and km/h -->
<p>This has   many   spaces.</p>
<p>This&nbsp;&nbsp;&nbsp;has&nbsp;&nbsp;&nbsp;many&nbsp;&nbsp;&nbsp;spaces.</p>

Result:

10 km/h

This has many spaces.

This   has   many   spaces.
