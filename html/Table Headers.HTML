HTML Table Headers, Footers, and Captions

HTML tables can be structured with headers, a body, and a footer section to provide better semantics and allow for more advanced styling and functionality (like scrolling the body independently of the header).

The <thead>, <tbody>, and <tfoot> Elements

  • The <thead> element is used to group header content in an HTML table.
  • The <tbody> element is used to group the body content in an HTML table.
  • The <tfoot> element is used to group footer content in an HTML table.

These elements do not affect the layout of the table by default, but they provide important semantic information and can be targeted by CSS and JavaScript.

Note: The <thead> must appear before <tbody> and <tfoot> within the <table> element.

Example of a Structured Table:

Monthly Savings Report
Month Savings
January $100
February $80
March $120
Total $300

The Code:

<table>
  <caption>Monthly Savings Report</caption>
  <thead>
    <tr>
      <th>Month</th>
      <th>Savings</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>January</td>
      <td>$100</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$80</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>$300</td>
    </tr>
  </tfoot>
</table>

The <th> Element (Table Header Cell)

The <th> element defines a cell as a header for a group of table cells. By default, the text in <th> elements is bold and centered (though this can be changed with CSS).

You can have headers for columns (most common) or for rows.

Vertical Headers (Row Headers)

Firstname: Henry
Lastname: Ford
Age: 80

The Code:

<tr>
  <th>Firstname:</th>
  <td>Henry</td>
</tr>

The <caption> Element

The <caption> tag defines a table caption. It must be inserted immediately after the <table> tag. By default, the caption is center-aligned above the table, but this can be changed with the CSS caption-side property.
