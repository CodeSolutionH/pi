HTML Tables

HTML tables are used to present data in a two-dimensional format of rows and columns.

Basic Table Structure:

<table>
  <tr>
    <th>Company</th>
    <th>Contact</th>
    <th>Country</th>
  </tr>
  <tr>
    <td>Alfreds Futterkiste</td>
    <td>Maria Anders</td>
    <td>Germany</td>
  </tr>
</table>

Table Elements:

  • <table>: The wrapper for the entire table.
  • <tr>: A table row.
  • <th>: A table header cell.
  • <td>: A table data cell.
