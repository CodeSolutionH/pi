HTML Text Formatting

HTML provides several elements for formatting text, such as <b> (bold), <i> (italic), <u> (underline), <mark> (highlight), and more.

Examples:

<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined Text</u>
<mark>Highlighted Text</mark>
<small>Small Text</small>
<del>Deleted Text</del>
<ins>Inserted Text</ins>
<sub>Subscript</sub>
<sup>Superscript</sup>

Rendered Output:

Bold Text
Italic Text
Underlined Text
Highlighted Text
Small Text
Deleted Text
Inserted Text
Subscript
Superscript
