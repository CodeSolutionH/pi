HTML Unordered Lists

An unordered list is a collection of items marked with bullets. Use the <ul> element for unordered lists, and <li> for each item.

Example:

<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
</ul>
  • Milk
  • Eggs
  • Bread

Attributes:

  • type (deprecated): Change bullet style (disc, circle, square).
  • Use CSS for modern styling: list-style-type.

Best Practices:

  • Use unordered lists for items where order does not matter.
  • Keep list items concise and related.
